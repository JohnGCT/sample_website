Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but the do list many of the most important ones.'

Document Elements

Tag Name Purpose
html HTML make a container for HTML elements
head Head make a container for metadata elements
body Body make a container for the HTML Document contents
title Title include a title for the HTML Document
meta Metadata make a container for metadata information of the HTML Document

Block Elements

Tag Name Purpose
header Header Make a header
div Division Make a division/container for HTML elements
h1–h6 Headings Include a heading (levels 1–6)
table Table Include a table
tr Table row Include a row of data
th Table header Make a table header
td Table data Include a table data cell
ol Ordered list Make an ordered list
ul Unordered list Make an unordered list
li List item Include a list item
blockquote Block quotation Show formatted quotation
br Break Enter line break

Inline Elements

Tag Name Purpose Example Result
em Emphasized Make emphasized text <em>technical sophistication</em> techincal sophistication
strong Strong Make strong text <strong>at least a billion people</strong> at least a billion people
a Anchor Make hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img Image Include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
span Span Include an inline container for HTML elements <span>Call me Ishmael</span> Call me Ishmael
